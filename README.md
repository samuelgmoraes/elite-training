# Elite Training ⚡

Landing page para uma consultoria de treino online. O projeto tem visual completo, simulação de pagamento, área do aluno e formulário de contato.

![Preview do projeto](assets/banner.jpg)

---

## O que tem na página

- Menu fixo no topo que escurece ao rolar, com versão mobile
- Seção principal com animações e botões de ação
- 4 cards explicando a metodologia de treino
- 6 depoimentos com avatar e resultado de cada aluno
- 3 planos de assinatura (Starter, Elite e VIP)
- Formulário de contato com validação de e-mail
- Footer com links de privacidade, termos e reembolso

---

## Pagamento simulado

Modal com 3 formas de pagamento:

- Cartão de crédito com máscara automática nos campos
- PIX com QR Code
- Boleto com instruções de pagamento

---

## Área do aluno

Página separada que simula o painel de quem assinou:

- Treinos da semana com status (feito, hoje, próximo)
- Barras de progresso do programa
- Campo de anotações salvo no navegador

---

## Tecnologias

- HTML, CSS e JavaScript
- [AOS.js](https://michalsnik.github.io/aos/) para animações
- Google Fonts (Montserrat)

---

## Responsividade

Funciona em mobile, tablet e desktop. No mobile o menu vira hamburguer e os cards se adaptam para coluna única.
